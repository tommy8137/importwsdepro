## 使用範例
### 在黑色背景的Modal裡面使用Button

<Playground>
    <div className={styles['display-button-theme']} style={{ background: '#333' }}>
        <Button border={false} color="transparentInModal">white</Button>
        <Button color="black">black</Button>
    </div>
</Playground>

### 按鈕顏色

<Playground>
    <div className={styles['display-button-theme']}>
        <Button >black</Button>
        <Button inverse>black (inverse)</Button>

        <Button color="white" inverse>white (inverse)</Button>

        <Button color="green">green</Button>
        <Button color="green" inverse>green (inverse)</Button>

        <Button color="transparent" border={false} width="auto">transparent</Button>
    </div>
</Playground>



### 按鈕形狀

    <div className={styles['display-button-theme']}>

</Playground>
